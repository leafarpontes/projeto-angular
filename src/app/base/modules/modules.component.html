Modules

Os módulos NgModules diferem e complementam os módulos JavaScript (ES2015). Um NgModule declara um contexto de compilação para um conjunto de componentes que é dedicado a um domínio de aplicativo, um fluxo de trabalho ou um conjunto de recursos intimamente relacionado. Um NgModule pode associar seus componentes a códigos relacionados, como serviços, para formar unidades funcionais.

Cada aplicativo Angular possui um módulo raiz, convencionalmente denominado AppModule, que fornece o mecanismo de bootstrap que inicia o aplicativo. Um aplicativo normalmente contém muitos módulos funcionais.

Como os módulos JavaScript, os NgModules podem importar funcionalidades de outros NgModules e permitir que sua própria funcionalidade seja exportada e usada por outros NgModules. Por exemplo, para usar o serviço de roteador em seu aplicativo, você importa o Router NgModule.

Organizar seu código em módulos funcionais distintos ajuda a gerenciar o desenvolvimento de aplicativos complexos e a projetar para reutilização. Além disso, essa técnica permite que você aproveite o carregamento lento - ou seja, o carregamento de módulos sob demanda - para minimizar a quantidade de código que precisa ser carregada na inicialização.

Criando um módulo usando o Angular CLI

  1. Em uma janela do terminal, navegue até o diretório da sua aplicação.
  2. Digite o comando: ng generate module <name>, onde name será o nome do módulo a ser criado.
