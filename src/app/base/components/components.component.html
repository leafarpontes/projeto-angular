Components

Os componentes são o bloco de construção principal em aplicações Angular, cada componente consiste em:

  • Um modelo HTML que declara o que é renderizado na página
  • Uma classe TypeScript que define o comportamento
  • Um seletor CSS que define como o comportamento é usado em um modelo
  • Opcionalmente, estilos CSS aplicados ao modelo

Pré-requisitos

Para criar um componente, verifique se você atendeu aos seguintes requisitos:

  1. Instale o Angular CLI
  2. Crie uma aplicação Angular, se você não tem um projeto, crie um usando ng new <nome-do-projeto>, onde <nome-do-projeto> é o nome da sua aplicação.

Criando um componente usando Angular CLI

  1. Em uma janela do terminal, navegue até o diretório da sua aplicação.
  2. Digite ng generate component <component-name>, onde <component-name> é o nome do seu componente.

Por padrão, esse comando cria o seguinte:

  • Uma pasta com o nome do componente
  • Um arquivo de componente, <component-name>.component.ts
  • Um arquivo de modelo, <component-name>.component.html
  • Um arquivo CSS, <component-name>.component.css
  • Um arquivo de testes, <component-name>.component.spec.ts

Onde <component-name> é o nome do seu componente.
